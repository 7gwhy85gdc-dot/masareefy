# مصاريفي 💵

تطبيق ويب (PWA) عربي لإدارة المصروفات الشخصية والميزانية الشهرية — يعمل بدون إنترنت، والبيانات محفوظة محليًا على جهازك فقط (LocalStorage). لا يوجد Backend ولا أي خدمات خارجية.

## التقنيات
React 18 + Vite + TypeScript + Tailwind CSS + Recharts، مع manifest.json و Service Worker ودعم كامل لـ RTL و safe-area في iPhone.

## التشغيل محليًا

```bash
npm install
npm run dev
```

ثم افتح العنوان الظاهر في الطرفية (عادة http://localhost:5173).

## التجربة على iPhone

1. تأكد أن الجوال والكمبيوتر على نفس شبكة الواي فاي.
2. شغّل `npm run dev` (الأمر مضبوط مع `--host` فيظهر عنوان Network مثل `http://192.168.1.x:5173`).
3. افتح هذا العنوان من Safari في الآيفون.

ملاحظة: Service Worker يعمل في الإنتاج فقط. لتجربة وضع Offline كاملًا:

```bash
npm run build
npm run preview
```

## الإضافة للشاشة الرئيسية في iPhone

1. افتح التطبيق من Safari.
2. اضغط زر المشاركة (المربع مع السهم).
3. اختر «إضافة إلى الشاشة الرئيسية».
4. سيظهر التطبيق بأيقونته وبدون شريط المتصفح (standalone) مع دعم النوتش و Dynamic Island.

## النشر

المشروع جاهز للنشر على Vercel أو Netlify مباشرة:
- Build command: `npm run build`
- Output directory: `dist`

## هيكل المشروع

```
src/
├── pages/            الصفحات الرئيسية (الرئيسية، التنبيهات، العروض، حسابي)
├── features/
│   ├── transactions/ العمليات + إضافة عملية (Bottom Sheet)
│   ├── budgets/      الميزانية الشهرية وميزانيات التصنيفات
│   ├── goals/        الأهداف المالية
│   └── analytics/    التحليلات والرسوم البيانية
├── components/
│   ├── ui/           مكونات عامة (Sheet, Toast, ProgressBar, ...)
│   └── layout/       Header, BottomNav, نظام التنقل
├── store/            المخزن (Context + Reducer + LocalStorage) والمحدِّدات
├── lib/              تنسيق الأرقام والتواريخ، الفترات المالية، البيانات التجريبية
└── types/            نماذج البيانات (Transaction, Budget, Goal, Category)
```

## استيراد كشف حساب الراجحي (PDF)

من صفحة «العمليات» (زر 🏦) أو «حسابي ← استيراد كشف حساب الراجحي»:

1. اختر ملف كشف الحساب PDF الأصلي من تطبيق الراجحي.
2. تتم القراءة كاملة داخل المتصفح (pdfjs) — لا يُرفع الملف لأي خادم.
3. تظهر معاينة: عدد العمليات، إجمالي المصروفات والإيداعات، مع تصنيف تلقائي ذكي (وقود، مطاعم، مقاهي، بقالة، تحويلات…) واسم المتجر أو الطرف الآخر للتحويل.
4. العمليات المكررة (نفس اليوم والمبلغ) تُستبعد تلقائيًا ويمكن إعادة تفعيلها.
5. يمكن استبعاد أي عملية قبل الاستيراد، أو إيقاف استيراد الإيداعات.

## ميزات النسخة الثانية

- سحب (Swipe) على أي عملية: تعديل، كرر، حذف — مع اهتزاز خفيف.
- اقتراح المتاجر السابقة تلقائيًا مع تذكّر تصنيف كل متجر.
- ملخص أسبوعي في الرئيسية مقارنة بالأسبوع الماضي.
- الالتزامات الشهرية المتكررة (صفحة الميزانية): تُسجَّل تلقائيًا في يومها، مع اكتشاف ذكي للمصروفات المتكررة واقتراح إضافتها.
- «أضف مبلغ للهدف» يحدّث الهدف ويسجل عملية ادخار معًا.
- «اسأل مصاريفك» في التحليلات: أسئلة عربية مثل «كم صرفت على المقاهي هذا الشهر؟».
- نظام الشهر الهجري 🌙 كخيار من حسابي.
- حماية البيانات: مرآة تلقائية في IndexedDB + تذكير شهري بالنسخ الاحتياطي + مشاركة نسخة عبر AirDrop/واتساب.

## ملاحظات

- عند أول تشغيل تُضاف بيانات تجريبية تلقائيًا — يمكن مسحها من: حسابي ← مسح جميع البيانات.
- بداية الشهر المالي قابلة للتغيير من صفحة حسابي.
- تصدير/استيراد نسخة احتياطية JSON من صفحة حسابي.
